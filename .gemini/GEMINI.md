# Doubles Plan Project Guidelines

## 1. 프로젝트 목표
- 1인용 심리 전략 카드 게임 <더블스 플랜>의 웹 브라우저 실행 환경 구축.
- GitHub Pages를 통한 정적 웹 사이트 배포 및 서비스.

## 2. 기술 스택
- 순수 Vanilla JavaScript (ES6+), HTML5, CSS3 사용 (필요시 경량 UI 프레임워크 적용).
- 서버사이드 렌더링이나 백엔드 연동 없이 브라우저 내에서 100% 로컬 로직으로 동작하도록 구현.

## 3. 핵심 게임 규칙 (구현 필수 사항)
- **컴포넌트**: 1~12 숫자 타일 (빨강: 3~9 / 검정: 1, 2, 10, 11, 12).
- **게임 구성**: 총 4세트 진행 (세트당 '블록 선택' 1회 + '숫자 대결' 5회).
- **블록 선택**: 선/후 플레이어 교대로 뒷면 색상만 보고 빨강 3개, 검정 2개 선택. 남은 2개는 비공개 폐기.
- **숫자 대결**: 선 플레이어 제출 -> 후 플레이어 확인 후 제출 -> 동시 공개.
- **승패 판정**:
  - 기본승: 높은 숫자 승리
  - 역전승: (높은 숫자) > (낮은 숫자) × 2 인 경우, **낮은 숫자 승리**
  - 더블승: (높은 숫자) == (낮은 숫자) × 2 인 경우, **높은 숫자 승리**
- **점수 산정**:
  - 1세트: 모든 승리 1점
  - 2세트: 기본/역전승 1점, 더블승 2점
  - 3, 4세트: 기본/역전승 2점, 더블승 4점
- **승리 조건**: 4세트 종료 후 합산 점수가 높은 쪽 승리. 동점 시 3/4세트 룰로 연장전.
