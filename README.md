# Doubles Plan (더블스 플랜)

## 📌 소개
웹 브라우저에서 실행되는 1인용 심리 전략 카드 게임입니다. 플레이어는 AI와 1:1로 숫자 블록을 제출하며 승리 조건(기본승, 역전승, 더블승)에 따라 점수를 획득합니다.

## 🚀 플레이 방법 (Play Now)
[https://7hehans.github.io/doubles_plan/]
*별도의 설치 없이 브라우저에서 바로 플레이 가능합니다.*

## 📜 핵심 게임 규칙
- **타일 구성:** 1~12 숫자 (빨강: 3~9 / 검정: 나머지)
- **진행:** 총 4세트 (블록 선택 및 5번의 숫자 대결)
- **승패 판정:**
  - 기본승: 높은 숫자 승리
  - 역전승: (높은 숫자) > (낮은 숫자) × 2 -> 낮은 숫자 승리
  - 더블승: (높은 숫자) == (낮은 숫자) × 2 -> 높은 숫자 승리
