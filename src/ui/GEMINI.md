# Web UI Implementation Guide

## 1. 뷰 렌더링 및 레이아웃
- 모바일과 데스크톱 모두에서 플레이 가능하도록 반응형 웹 디자인(Responsive Web Design) 적용.
- 타일의 색상(빨강/검정)을 시각적으로 명확히 구분할 수 있도록 CSS 디자인 구성 (카드 형태의 UI).
- 자신의 타일은 앞면(숫자)이 보이게, 상대방(AI)의 타일은 뒷면(색상)만 보이게 렌더링.

## 2. 인터랙션 및 애니메이션
- 타일 선택 및 제출 시 드래그 앤 드롭(Drag & Drop) 또는 클릭 이벤트를 지원.
- 대결 페이즈에서 양측 타일 공개 시 딜레이 애니메이션을 주어 긴장감 연출.

## 3. 정보 시각화
- 현재 세트, 진행 라운드, 양측의 누적 점수, 남은 타일의 색상 분포를 화면 상단에 상시 표기.
- 승패 판정 결과(기본승/역전승/더블승)를 토스트 메시지나 모달 팝업으로 즉각 피드백.
